To top up your DeepSeek balance, log into the DeepSeek Platform and navigate to the Billing or Top Up page on the dashboard. You can add funds on a pay-as-you-go basis (in USD or CNY) using PayPal, bank cards, Alipay, or WeChat Pay.

### How the Billing System Works

Pay-as-you-go: It is not a monthly subscription. You add funds (e.g., $5 or $10) which are only deducted as you consume tokens for API requests. 

Balance Expiry: Any topped-up balance does not expire. If you have "granted" (promotional) balances, those may have an expiration date.Usage 

Preference: When making API calls, your granted balance is used first, followed by your topped-up balance.Refunds: Unused, topped-up balances are refundable directly from the Billing page.
