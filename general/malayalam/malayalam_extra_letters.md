Below is a beginner-friendly study guide for **Malayalam Chillaksharangal, Chihnangal, and Koottaksharangal**.

Malayalam vowels and consonants are fairly easy to learn first. The difficulty usually starts with these three areas:

**Chillaksharangal** = special “pure consonant” letters
**Chihnangal** = vowel signs added to consonants
**Koottaksharangal** = joined consonant letters / conjunct letters

---

# 1. Chillaksharangal — ചില്ലക്ഷരങ്ങൾ

**Chillaksharam** means a consonant that stands alone without the vowel sound **“a”**.

Example:
**ന** is pronounced **na**
**ൻ** is pronounced only **n**

So **ൻ, ൽ, ൾ, ൺ, ർ, ൿ** are special final consonant forms.

| Chillaksharam | English sound | Malayalam word | Transliteration | Meaning          |
| ------------- | ------------- | -------------- | --------------- | ---------------- |
| ൻ             | n             | ഞാൻ            | ñān / njān      | I                |
| ൽ             | l             | കാൽ            | kāl             | leg / foot       |
| ൾ             | retroflex l   | വാൾ            | vāḷ             | sword            |
| ൺ             | retroflex n   | മൺ             | maṇ             | soil             |
| ർ             | r             | കാർ            | kār             | car              |
| ൿ             | k             | rarely used    | —               | mostly old usage |

## How to understand them

| Normal letter | Sound | Chillu form | Sound |
| ------------- | ----- | ----------- | ----- |
| ന             | na    | ൻ           | n     |
| ല             | la    | ൽ           | l     |
| ള             | ḷa    | ൾ           | ḷ     |
| ണ             | ṇa    | ൺ           | ṇ     |
| ര             | ra    | ർ           | r     |
| ക             | ka    | ൿ           | k     |

### Practice words

# 2. Chihnangal — ചിഹ്നങ്ങൾ

**Chihnangal** are vowel signs.
In Malayalam, a consonant normally contains the sound **“a”**.

Example:
**ക = ka**

To change **ka** into **ki, ku, ke, ko**, etc., we add vowel signs.

| Vowel sign | Letter form | Sound      | Transliteration | Example word                   |
| ---------- | ----------- | ---------- | --------------- | ------------------------------ |
| —          | ക           | ka         | ka              | കട — kaṭa                      |
| ാ          | കാ          | kaa        | kā              | കാട് — kāṭu                    |
| ി          | കി          | ki         | ki              | കിളി — kiḷi                    |
| ീ          | കീ          | kee        | kī              | കീടം — kīṭam                   |
| ു          | കു          | ku         | ku              | കുട — kuṭa                     |
| ൂ          | കൂ          | koo        | kū              | കൂട് — kūṭu                    |
| ൃ          | കൃ          | kru / kṛ   | kṛ              | കൃപ — kṛpa                     |
| െ          | കെ          | ke         | ke              | കേരളം — kēraḷam starts with കേ |
| േ          | കേ          | kay / kē   | kē              | കേൾക്കുക — kēḷkkuka            |
| ൈ          | കൈ          | kai        | kai             | കൈ — kai                       |
| ൊ          | കൊ          | ko         | ko              | കൊടി — koṭi                    |
| ോ          | കോ          | kō         | kō              | കോടി — kōṭi                    |

# 3. Koottaksharangal — കൂട്ടക്ഷരങ്ങൾ

**Koottaksharangal** are joined consonants.

In English, we write consonant clusters separately:

**school** = s + c + h + o + o + l
**plant** = p + l + a + n + t

In Malayalam, consonants often join into a combined form.

Example:
**ക + ക = ക്ക**
This is pronounced **kka**

| ഞ്ച           | ഞ + ച             | ncha / ñcha   | അഞ്ചു          | añchu           | five             |

# 5. Some Malayalam sounds that are hard for English speakers

Malayalam has several sounds that English does not clearly distinguish.

## Dental vs retroflex sounds

| Malayalam | Approx English sound        | Example | Transliteration |
| --------- | --------------------------- | ------- | --------------- |
| ത         | soft tha, tongue near teeth | തല      | thala           |
| ട         | hard ṭa, tongue curled back | പട്ടം   | paṭṭam          |
| ന         | na                          | നരി     | nari            |
| ണ         | ṇa, tongue curled back      | മണി     | maṇi            |
| ല         | la                          | മല      | mala            |
| ള         | ḷa, tongue curled back      | വള      | vaḷa            |
| ര         | soft ra                     | രവി     | ravi            |
| റ         | stronger ra / ṟa            | മുറി    | muṟi            |

For beginners, it is okay to first learn them approximately. Later, improve the tongue position.

# 6. Beginner learning order

Use this order:

## Step 1: Learn normal consonants and vowels

Example:

ക, ഖ, ഗ, ച, ജ, ട, ത, പ, മ, യ, ര, ല, വ, ശ, സ, ഹ

Then vowels:

അ, ആ, ഇ, ഈ, ഉ, ഊ, എ, ഏ, ഐ, ഒ, ഓ, ഔ

## Step 2: Learn vowel signs using one consonant

Practice with **ക**:

ക, കാ, കി, കീ, കു, കൂ, കെ, കേ, കൈ, കൊ, കോ, കൗ

Then repeat with:

മ, ന, പ, ല, ര

Example:

മ, മാ, മി, മീ, മു, മൂ, മെ, മേ, മൈ, മൊ, മോ

# 8. Simple rule to remember

Malayalam reading becomes easier when you understand this:

**Consonant alone usually has “a” sound.**
ക = ka
മ = ma
ന = na

**Chihnam changes the vowel.**
കി = ki
കു = ku
കോ = kō

**Chillu removes the vowel.**
ന = na
ൻ = n

**Koottaksharam joins consonants.**
ക + ക = ക്ക = kka
മ + മ = മ്മ = mma
